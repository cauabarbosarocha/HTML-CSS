Principais formatações

Negrito / Destaque

Nessa frase temos um termo em destaque usando a tag <b> (não semântica).
Nessa frase temos outro termo em destague usando a tag <strong>(semântica).

Itálico / Ênfase

Nesta frase, temos um termo itálico usando a tag <i> (não semântica).
Nesta frase, temos um termo em ênfase usando a tag <em> (semântica)

Texto marcado

Podemos criar também um texto marcado usando a tag <mark>.
E aqui tem outro texto marcado.
Utilizando a tag <mark>

Textos grandes e pequenos

Estamos criando um texto grande e um texto pequeno neste parágrafo.
Utilizando a tag <big> para deixar grande, e a tag <smal> para deixar grande.

Texto deletado

Podemos marcar um texto como excluído para indicar que ele deve ser lido mas não considerado.
Utilizando a tag <del>

Texto inserido

Podemos marcar um texto como inserido para dar uma ênfase e indcar que ele foi adicionado depois.
Utilizando a tag <ins>, ou com a tag <u> (não semântica)

Texto sobrescrito

Para inserir coisas do tipo x20+3.
Utilizando a tag <sup>

Texto subscrito

Para inserir coisas do tipo H2O.
Utilizando a tag <sub>
